  • Logic Gate Tool Project

    A website that can simulate simple logic circuits. Collaboratively made by future engineers Prince Adeyemi Rasheed, Jan Adams Gempes, and Billy Joe Ivan Carde√±o, in fulfillment of a semester-end project in CPEN116 (Online Technology) headed by Mrs. Eileen Rocillo.

    How-to:

    • Click on a component on the left then click on the canvas to place it.
    • Components can be linked by clicking on either the input or output nodes of the target components.
    • The first component is a logic input, switch it on/off by double clicking it.
    • The second component indicates the logic output.
